# StudioAgent - Glossario

**Purpose:** Mantenere terminologia consistente tra CMO, CTO e documentazione.

---

## Termini Italiani (Settore Condominio)

| Termine | Significato | Esempio d'uso |
|---------|-------------|---------------|
| **Condominio** | Edificio/complesso abitativo con più unità immobiliari | "Gestiamo 150 condomìni" |
| **Amministratore** | Professionista che gestisce uno o più condomìni | "L'amministratore medio perde 5h/giorno" |
| **Condòmino** | Proprietario di un'unità immobiliare nel condominio | "Il condòmino chiede info su spese" |
| **Studio** | Ufficio professionale dell'amministratore (nostro cliente) | "Target: studi 20-80 condomìni" |
| **Unità immobiliare** | Singolo appartamento/negozio nel condominio | "Condominio con 45 unità" |
| **Assemblea** | Riunione annuale/straordinaria dei condòmini | "Convocazione assemblea ordinaria" |
| **Regolamento condominiale** | Documento che regola vita condominiale | "Upload regolamento per RAG" |
| **Millesimi** | Quote di proprietà nel condominio | "Spese ripartite per millesimi" |
| **Supercondominio** | Complesso di più edifici con servizi comuni | "Gestione supercondominio 300+ unità" |

## Termini Prodotto (StudioAgent)

| Termine | Significato | Note |
|---------|-------------|------|
| **Dashboard** | Interfaccia principale admin/operatore | Non "pannello di controllo" |
| **Knowledge Base** | Database normativa + regolamenti condominiali | Per operatori studio, non condòmini |
| **Chat Module** | Interfaccia unificata WhatsApp + Email | Non "messaggistica" |
| **AI Engine** | Sistema generazione risposte automatiche | Claude + RAG su normativa |
| **Operatore** | Segretaria/collaboratore dello studio | Ruolo utente con permessi limitati |
| **Admin Studio** | Titolare/responsabile studio (cliente) | Ruolo utente con full access |
| **Template** | Risposta pre-compilata frequente | Es. "convocazione assemblea" |
| **RAG** | Retrieval-Augmented Generation | Tecnica AI per risposte contestuali |
| **Workflow** | Sequenza automatizzata azioni | Es. "auto-reply fuori orario" |

## Termini Business

| Termine | Significato | Note |
|---------|-------------|------|
| **MVP** | Minimum Viable Product | Prima versione funzionale prodotto |
| **MRR** | Monthly Recurring Revenue | Entrate mensili ricorrenti |

| **Burn Rate** | Velocità consumo budget | €/mese spesi |
| **Runway** | Tempo rimanente prima di cash-out | Mesi con budget attuale |

## Evitare (Termini Vietati)

| ❌ Non usare | ✅ Usare invece | Perché |
|-------------|----------------|--------|
| "Chatbot" | "AI Assistant" o "Knowledge Base" | "Chatbot" = low-quality, spam |
| "Rivoluzionare" | "Semplificare" o "Automatizzare" | Target conservatore, evitare hype |
| "Disruption" | "Innovazione" | Anglicismo inutile |
| "Paradigma" | Parola semplice specifica | Buzzword vuota |
| "Engagement" | "Coinvolgimento" | Preferire italiano quando possibile |
| "Machine Learning" | "Intelligenza Artificiale" | ML troppo tecnico per target |
| "Suite" (nel nome) | "Piattaforma" o "Sistema" | Nel nome prodotto ok, in copy evitare |

## Abbreviazioni Standard

| Abbreviazione | Significato | Quando usare |
|--------------|-------------|--------------|
| **AI** | Artificial Intelligence | Ok in documentazione tecnica |
| **IA** | Intelligenza Artificiale | Preferire in comunicazione cliente italiano |
| **API** | Application Programming Interface | Ok in contesto tech |
| **CRM** | Customer Relationship Management | Ok, target lo conosce |
| **SaaS** | Software as a Service | Ok in business plan, evitare in marketing copy |
| **UX** | User Experience | Ok interno, evitare con clienti |
| **UI** | User Interface | Ok interno, evitare con clienti |
| **MVP** | Minimum Viable Product | Solo uso interno |
| **GTM** | Go-to-Market | Solo uso interno |

## Note Stilistiche

**Tone italiano:**
- ✅ "Risparmiare tempo" (diretto, chiaro)
- ❌ "Ottimizzare il workflow operativo" (pomposo, vago)

**Numeri:**
- ✅ "5 ore al giorno" (specifico)
- ❌ "Tanto tempo" (vago)

**Benefici:**
- ✅ "Rispondi ai condòmini in 30 secondi invece di 30 minuti"
- ❌ "Migliora l'efficienza della comunicazione"
